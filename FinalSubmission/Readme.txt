Project Title : Predicted Energy Usage in a household

Description:

This project focuses on predicting global active power consumption using machine learning models. It encompasses data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation for accurate predictions.

Requirements
Software and Libraries
Python 3.8+
Libraries:
pandas, numpy, matplotlib, seaborn, plotly
scikit-learn, xgboost, lightgbm, prophet, statsmodels
